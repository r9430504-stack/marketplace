# Фото телефонов

Сюда кладутся реальные фотографии моделей. Имя файла = `slug` модели
(его список — в `src/data/phones.ts`), напр.:

    public/models/galaxy-s24-ultra.jpg
    public/models/galaxy-z-flip-6.webp

Затем в `src/data/phones.ts` у нужной модели указывается поле:

    image: "/models/galaxy-s24-ultra.jpg",

После этого карточка автоматически показывает фото вместо заглушки.

## Требования к фото
- Только изображения, на которые есть права (свои снимки или свободные лицензии).
- Официальные пресс-фото Samsung без лицензии использовать нельзя.
- Желательно вертикально, соотношение около 3:4, фон однотонный/прозрачный.
- Формат: JPG, PNG или WebP. Оптимально до ~300 КБ.
